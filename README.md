# Park&Go

## Introduction
Commuting to a large campus such as the University of Minnesota can be challenging, particularly when it comes to finding convenient parking. Park&Go is a web application designed to help University of Minnesota students locate available parking options quickly and efficiently. The platform provides multiple choices, including street parking, contract parking, and campus garages, enabling users to compare and select the most suitable option. Park&Go simplifies the parking experience and makes getting to campus easier.

## Version 1 — 12/31/2025
This initial release includes the front-end implementation of the application. Future versions will introduce a back-end service that allows users to interact with live parking options and selections.

## Technology Used
- HTML
- CSS
- JavaScript

## Features
- Search and compare parking options
- View garage and contract availability
- Mobile-friendly interface
- Quick navigation links to campus locations
- User-submitted parking suggestions

## Future Enhancements
- Implement a back-end service with predefined parking locations that users can select based on their current area
- Enhance the suggestion feature to predict the shortest route using a route-prediction system
- Add hard-coded map markers and routing visuals for improved navigation
